Wolf Adventure
自作の素材（グラフィック・BGM）にこだわった、オーソドックスな2D横スクロールアクションゲームです。

1. 概要
プレイヤーを操作し、道中でてくる敵を倒したり避けたりしながら、ステージ最奥のゴールを目指します。 アクションの基本である操作の心地よさと、敵キャラクターの挙動などを意識して制作しました。

開発期間: 約3か月

プラットフォーム: PC (Windows)

制作範囲: プログラム、全グラフィック（キャラ・背景）、BGM等制作
効果音：効果音ラボ様 https://soundeffect-lab.info/

2. 使用技術
言語: C#

ゲームエンジン: Unity 2022.3.21f1 

ツール: Visual Studio Code、Studio One 6 Artist(作曲)、EDGE(ドット絵)

3. 技術的なこだわり・工夫した点
① 接地判定の最適化と操作感の向上
プレイヤーのジャンプアニメーションと操作を正確に制御するため、物理エンジン（Rigidbody2D）に頼り切るのではなく、足元に専用の小さな当たり判定（TouchingDirections.cs）を配置しました。

メリット: 坂道での意図しないジャンプ不可状態を防ぎ、着地した瞬間に次のアクションへ遷移できるキレのある操作感を実現しました。

② 敵キャラクターの「落下防止」AI
敵がただ往復するだけでなく、知的な挙動に見えるよう工夫しました。

実装: 進行方向の足元にレイ（Raycast）や判定エリアを飛ばし、「この先が崖であるか」を事前に検知する仕組みを導入しました。

結果: 敵が不自然にステージ外へ落下するのを防ぎ、ゲームとしての難易度と品質を維持しました。

③ 全素材の完全オリジナル制作
ゲーム体験の統一感を出すため、全てのビジュアルと音声を自作しました。

グラフィック: 世界観に合わせたドット絵（またはイラスト）を制作。

サウンド: ステージの雰囲気に合わせたBGMを構成し、ゲームへの没入感を高めました。

4. 操作方法
移動: A / D キー

ジャンプ: Space キー

攻撃: K キー（例）

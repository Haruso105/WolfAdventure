# Wolf Adventure
自作の素材（グラフィック・BGM）にこだわった、オーソドックスな2D横スクロールアクションゲームです。

## 概要
プレイヤーを操作し、道中でてくる敵を倒したり避けたりしながら、ステージ最奥のゴールを目指します。 アクションの基本である操作の心地よさと、敵キャラクターの挙動などを意識して制作しました。

開発期間: 約3か月

プラットフォーム: PC (Windows)

制作範囲: プログラム、全グラフィック（キャラ・背景・UI）、BGM等制作

効果音：効果音ラボ様 https://soundeffect-lab.info/

本プロジェクトで作成したソースコードは、以下のディレクトリからご確認いただけます。
> **ソースコード一覧:** [Assets/Scripts](./Assets/Scripts)

### ジャンプ中・空中の操作感の向上
プレイヤーのジャンプ操作・空中操作を自然な動きにするため、物理エンジン（Rigidbody2D）に頼り切るのではなく、ジャンプの飛距離に応じた速度減衰の処理を記述しました。
(Assets/Scripts/PlayerController.cs.)


### 敵キャラクターの「落下防止」
敵がただ往復するだけでなく、知的な挙動に見えるよう工夫しました。

実装: 進行方向の足元にRaycastや判定エリアを飛ばし、「この先が崖であるか」を事前に検知する仕組みを導入しました。敵が不自然に段差を落下するのを防ぎ、ゲームとしての難易度と品質を維持しました。
(Assets/Scripts/TouchingDirections.cs.)

### 全素材の完全オリジナル制作
ゲーム体験の統一感を出すため、全てのビジュアルと音声を自作しました。

グラフィック: 世界観に合わせたドット絵を制作。
サウンド: ステージの雰囲気に合わせたBGMを構成し、ゲームへの没入感を高めました。

### 操作方法
移動: A / D キー

ジャンプ: Z キー

攻撃: X キー
